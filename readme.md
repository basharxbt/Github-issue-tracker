2. What is the spread operator (...)?

Answer: The spread operator is used to expand elements of an array or object into individual elements

Example:
const arr = [1, 2, 3];
console.log(...arr);

output: 1 2 3

3. What is the difference between map(), filter(), and forEach()?

Answer: The difference between map(), filter(), and forEach() is how they process arrays and what they return.

map(): loops through an array and creates a new array by changing each element.

filter(): loops through an array and returns a new array with elements that match a condition.

forEach(): loops through an array but does not return a new array. It is used to perform an action.
